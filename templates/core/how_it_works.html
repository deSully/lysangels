Guide d'utilisation

Simple.
Honnête. Direct.

LysAngels est un annuaire de mise en relation. Vous trouvez le prestataire, vous le contactez directement. Pas de messagerie interne, pas de réservation en ligne.

Pour les clients

Trouver un prestataire
en 3 étapes

Aucun compte à créer. Aucune réservation en ligne. Juste la mise en relation.

1

Parcourez les profils

Utilisez les filtres pour trouver le prestataire qui correspond à votre événement : type de service, ville, disponibilité. Consultez les photos, les descriptions et les tarifs indicatifs. Aucun compte requis.

Voir les prestataires
2

Contactez directement

Sur chaque profil, vous trouverez les coordonnées du prestataire : numéro WhatsApp, téléphone. Vous l'appelez ou lui envoyez un message directement. La suite se passe hors plateforme.

Contact via WhatsApp ou téléphone

Les coordonnées sont affichées directement sur le profil du prestataire. Un clic suffit pour ouvrir WhatsApp.

3

Organisez votre événement

Vous négociez les détails, le prix et les conditions directement avec le prestataire, par téléphone ou en personne. LysAngels n'intervient pas dans cette étape. Vous êtes libres.

Vous préférez qu'on s'en occupe ?

Vous pouvez aussi déposer votre projet et laisser notre équipe vous proposer des prestataires adaptés. Ce service d'accompagnement est payant.

Pour les prestataires

Comment vous êtes contacté

Votre profil est votre vitrine. Les clients vous contactent directement — vous ne recevez rien sur la plateforme.

Votre profil est votre vitrine

L'équipe LysAngels crée et gère votre profil. Photos, description, services, tarifs indicatifs, coordonnées de contact. Vous n'avez pas accès à un tableau de bord.

Les clients vous appellent directement

Un client intéressé par votre profil vous contacte via votre numéro WhatsApp ou téléphone. Vous ne recevez aucune notification sur la plateforme.

Vous gérez votre activité

Vous négociez, signez et encaissez directement avec le client. LysAngels vous apporte de la visibilité — la relation commerciale reste entre vous et votre client.

Vous souhaitez apparaître sur LysAngels ?

Les profils sont créés et gérés par notre équipe. Contactez-nous pour soumettre votre candidature.
